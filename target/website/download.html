系统要求

OmegaT will run on any system on which the appropriate JRE (Java Runtime Environment) has been or can be installed. OmegaT 会自带 JRE，不需要单独下载。OmegaT 已在 Windows（除 98 外的所有版本）、Micintosh OS X 及 Linux 上成功安装。

Microsoft Office XML 格式的文件可直接在 OmegaT 中翻译。旧的 MS Word、Excel 和 Powerpoint 格式的文件可以在转换到等效的 Microsoft Office 格式文件后翻译，或者也可使用 OpenOffice.org 或 LibreOffice（同样免费）中转换成 Office Open XML 格式。获取 OpenOffice.org 或 LibreOffice，请点击左方“第三方软件”按钮链接到相应网站。

包含/不包含 JRE

要想运行 OmegaT，电脑上必须安装 Java 运行环境（JRE）。

如果您确定电脑中已经安装了适用的 JRE，您可下载不包含 JRE 的 OmegaT 版本。下载会更快，安装占用硬盘空间更小。在此情况下，请勿选择默认版本；选择不同版本（例如标准或最新版本），然后选择标记“不包含 Java”的版本。

如果您在电脑上已安装 JRE 的情况下下载了包含 JRE 的 OmegaT 版本，OmegaT 的 JRE 会独立安装，不会与已有的 JRE 冲突因此，安装含有 JRE 的 OmegaT 总是安全的。如果您不确定您是否已安装了适用的 JRE 版本，选择包含 JRE 的版本（或默认版本）。

Mac OS X 用户：Mac OS X 上已默认安装 JRE；因此，不提供 Mac OS X 上的包含 JRE 版本。

Linux 用户：多数 Linux 发行版都提供了 Java 运行环境，但一些开源实现不能很好地处理 OmegaT（尽管可能并不明显）。如果您是 Linux 的新用户，我们建议您选择包含 JRE 的 OmegaT 版本。较有经验的 Linux 用户可能更喜欢在系统上按照 Oracle（以前的 Sun）JRE。

已签名/未签名

Mac OS X 有一个叫做 Gatekeeper 的功能，该功能通过增加运行未签字程序的难度来增加安全性。Mac OS X 上的 OmegaT 签名和未签名版都可用；他们在功能上相同，但首选签名版本，因为其是由已知开发者发布的。
