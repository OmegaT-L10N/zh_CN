分词器插件已集成到3.0.0版的OmegaT中。以下信息仅当你使用早于3.0.0的OmegaT版本时才适用。
如果你曾经使用OmegaT的早期版本的分词器插件，并现在已升级到OmegaT的3.0.0版或更高版本，请从插件文件夹中删除分词器插件文件。

OmegaT tokenizer是OmegaT的一个插件。它通过计算源词的词根（“词干”）来为OmegaT提供更好的模糊匹配和术语表匹配。例如，即使词汇表条目仅包含单词的原型，它也会识别文本中的变形词并显示相应的词汇表条目。

准备

在使用OmegaT的分词器之前，你必须首先确保你的OmegaT版本适配分词器，并已经做好了准备。

Webstart版本的OmegaT:：分词器与Webstart版本的OmegaT不兼容。如果你希望使用分词器，请为你的系统安装标准版本的OmegaT（最新测试版）。

OmegaT 2.1.0 及之前版本：当前的分词器不适配这些版本（分词器可以与版本2.0.x和2.1.0一起使用，但这需要不同版本的分词器以及不同的安装过程。）建议用户升级到OmegaT的最新测试版。

Windows版本的 OmegaT：为了使OmegaT（任何版本）可以与分词器一起使用，它必须从启动脚本文件启动。Windows 版本的OmegaT不提供启动脚本文件。如果你使用的是有JRE环境的Windows，下载名为OmegaT_with_JRE.bat的文件; 如果你使用的是没有JRE环境的Windows， 下载名为OmegaT_without_JRE.bat的文件. 下载完成后，将文件放在OmegaT主文件夹（包含文件OmegaT.jar的文件夹）中。

平台中立版本（Windows系统）: 找到OmegaT启动脚本文件（OmegaT.bat）。

Linux 版本/系统：找到你的OmegaT启动脚本文件（OmegaT或OmegaT.sh）。

在执行启动脚本文件时检查是否启动了OmegaT
-在Linux系统上，使用命令行
- 在Windows系统上，点击启动脚本文件

安装分词器

做好安装准备工作之后（见上文），按照如下步骤安装分词器：

1. 下载 分词器压缩包 (OmegaT 2.1.1 及之后版本适用）.

2. 从分词器压缩包中解压文件

3. 在OmegaT主程序文件夹（即包含文件OmegaT.jar的文件夹）中，如果尚未存在名为“plugins”的子文件夹，则创建一个名为“plugins”的子文件夹。把你从分词器压缩包中解压缩的文件直接复制到此子文件夹中。

4. 在文本编辑器中打开启动脚本文件。（特别是）Windows用户： 不要 直接点击此文件。相反，启动文本编辑器（例如记事本或写字板），然后使用“文件>打开”打开启动脚本文件。你也可以在文件上单击鼠标右键，然后选择一个文本编辑器打开它。

5. 启动文件脚本包含OmegaT启动命令。该命令的基本形式为：

java -jar OmegaT.jar

根据您的系统配置，启动命令可能会稍有不同。

org.omegat.plugins.tokenizer.LuceneDutchTokenizer
org.omegat.plugins.tokenizer.LuceneFrenchTokenizer

org.omegat.plugins.tokenizer.LuceneGreekTokenizer

org.omegat.plugins.tokenizer.LuceneSmartChineseTokenizer
org.omegat.plugins.tokenizer.LuceneRussianTokenizer
org.omegat.plugins.tokenizer.LuceneThaiTokenizer
org.omegat.plugins.tokenizer.SnowballDanishTokenizer

在你的OmegaT启动脚本文件中将参数-- ITokenizer =后面加该分词器的全名（复制整行，并插入一个空格）添加到启动命令的末尾

例如，要使用英语分词器（从英语翻译成别的语言时），你的启动命令现在可能是：

java -jar OmegaT.jar %* --ITokenizer=org.omegat.plugins.tokenizer.SnowballEnglishTokenizer

或者，当你要从土耳其语翻译成别的语言时，你的启动命令现在可能是：

java -jar OmegaT.jar %* --ITokenizer=org.omegat.plugins.tokenizer.SnowballTurkishTokenizer

注意：这整个命令必须是一行（即使它在你正在阅读的显示器上可能显示为两行）。

7. 执行这个文件，现在再打开OmegaT ，应该就能使用分词功能了。若要验证这一点，请检查在当前OmegaT片段中包含术语表中词汇的变形的情况下，词汇表术语是否仍能显示。

8. 如果你要从多种源语言进行翻译，因此你希望使用不同的分词器，请为你希望使用的每个分词器创建单独的OmegaT启动脚本文件。为启动脚本文件进行合适的命名，例如，把包含英语分词器命令的启动脚本文件命名为“OmegaT-EN.bat”,把包含土耳其语分词器命令的启动脚本文件命名为“OmegaT-TR.bat”

9. 在某些情况下，你可能会发现源语言标记器会干扰目标语言拼写检查器。你可以通过也为目标语言指定一个分词器（如果可用），并使用参数--ITokenizerTarget =来消除此问题。

例如，如果你要从汉语翻译成荷兰语，尝试：

java -jar OmegaT.jar %* --ITokenizer=org.omegat.plugins.tokenizer.LuceneChineseTokenizer --ITokenizerTarget=org.omegat.plugins.tokenizer.LuceneDutchTokenizer

10. 在如上所述创建启动脚本之后，您可以配置你的系统以便更方便地启动OmegaT，例如创建快捷方式。在Windows中创建快捷方式：

右键单击启动脚本（OmegaT.bat），然后按住鼠标右键，将脚本拖到方便的位置，例如桌面。当你释放右键时，会打开一个对话框，其中包含多个选项。选择“在此创建快捷方式”。

或者，右键单击启动脚本。选择：“发送到”，然后选择“桌面（创建快捷方式）”。

在创建并测试快捷方式之后，你可以将其拖动到“开始”菜单，这样就可以将其添加到“开始”菜单中。
